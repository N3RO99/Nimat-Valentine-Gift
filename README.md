## Customization

### Change the Letter
Edit the text in `src/App.jsx` in the letter modal section.

### Add Images/Videos to Teddy Bear Gallery
Replace the image URLs in the teddy bear section or add video elements like:

```jsx
<video width="200" height="200" controls>
  <source src="your-video.mp4" type="video/mp4">
</video>
```

### Change the 14 Reasons
Edit the `reasons` array in `src/App.jsx` to customize the reasons and emojis.
